Pertemuan kali ini kita akan mambahas tentang Constant

Constant, sama seperti variabel yang digunakan untuk menyimpan suatu nilai yang dimana Constant itu tidak dapat dirubah nilainya sampai program selesai dijalankan

Ada 2 cara untuk membuat constant pada PHP, yaitu :
1. define(), tidak dapat digunakan di dalam class (global)
2. const, sebaliknya dari define, const dapat digunakan di dalam class

Magic Constant:
1. __LINE__
2. __FILE__
3. __DIR__
4. __FUNCTION__
5. __CLASS__
6. __TRAIT__
7. __METHOD__
8. __NAMESPACE__
