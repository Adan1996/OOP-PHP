Pertemuan kali ini kita akan mambahas tentang Visibility

Visibility, konsep yang digunakan untuk mengatur akses dari property dan method pada sebuah object.

Ada 3 keyword visibility, yaitu :
1. Public, dapat digunakan dimana saja bahkan diluar class
2. Protected, hanya dapat digunakan di dalam sebuah class beserta turunannya
3. Private, hanya dapat digunakan di dalam class sebuah tertentu saja

Kenapa harus menggunakan visibility ?
1. Hanya memperlihatkan aspek dari class yang dibutuhkan oleh client
2. Menentukan kebutuhan yang jelas untuk object
3. Memberikan kendali pada kode untuk menghindari 'bug'
