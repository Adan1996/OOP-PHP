Pertemuan kali ini kita akan mambahas tentang Static Keyword

Static Keyword, digunakan untuk mengakses property dan method dalam konteks class

untuk apa static keyword itu ?
1. Member yang terkait dengan class, bukan dengan object
2. Nilai static akan selalu tetap meskipun object diinstansiasi berulang kali
3. Membuat kode menjadi 'Procedural'
4. Biasanya digunakan untuk membuat fungsi bantuan / helper
5. Atau digunakan di class-class utility pada framework
